01 — Introduction

はじめまして。

秋月 遼（あきづき りょう）と申します。関西を拠点に、医療・介護分野の事業立ち上げと採用設計を手がけています。

表に出る仕事ではありません。経営者の隣に座り、現場の声に耳を傾け、人と仕組みを一緒に整える。そういう役回りで、気づけば十年が経っていました。

プロフィールは、素敵な見た目よりも、読んだ方が「ああ、こういう人か」と少しだけ掴めることの方が大切だと思っています。肩に力を入れず、いつもの速度で書きました。

02 — Background

生まれ育ち、家族のこと。

生まれは兵庫県、西播磨の小さな町です。周囲は山と田んぼ。人口よりも鹿のほうが多いような場所でした。いまは大阪に居を移していますが、夏になると実家の裏山の匂いを思い出します。

父は市役所、母は看護師。三人きょうだいの真ん中で、兄と妹の間で「ちゃんとしていなければ」という感覚が早くから育ちました。家庭の中に医療職が一人いる、というのは、思っているよりも子どもの進路に影響します。

「家族の頑張りだけに頼る介護は、
どこかで限界が来る。」

03 — Why This Work

なぜ、この仕事を選んだのか。

高校二年の夏、祖母が脳梗塞で倒れました。在宅で介護することになり、母は夜勤の合間に祖母の体位を変え、父は朝五時に起きて朝食の介助をしていました。私は、救急車を呼んだあの朝の手の震えを、いまでもはっきり覚えています。

祖母は半年後に亡くなりました。葬儀のあと、母が台所で静かに泣いていました。悲しみとはまた別の疲れが、その背中にありました。

社会人になって医療や介護の現場に触れるたびに、あの台所の景色が重なります。在宅の環境を、家族の献身だけに委ねきらない。そのための仕組みを、経営の側から整えたい。この仕事を続けている理由は、結局はそこに尽きます。

05 — Principles

仕事で大切にしている、四つのこと。

現場から、はじめる。

経営者の椅子からではなく、廊下を歩く人の視点から設計を始める。最初の三回は、必ず現場に入らせてもらう。

採用は、辞めない設計。

人を集めることよりも、入った人が二年、三年と残れる環境を作ることに時間を使う。結局はそれが一番、数字に効く。

短期の数字と、長い納得。

来期の経営計画を仕上げながら、十年先の理事長の心残りの方も、同じテーブルに置いて話す。

家族という変数を、忘れない。

利用者には家族がおり、職員にも家族がいる。その背景を落とさないことが、医療介護の仕事の品質を決める。

06 — Work

いま、お受けしている仕事。

基本的には、経営者の隣に座って中長期で関わる仕事をしています。単発の研修や、その場限りのアドバイスは、あまり向いていないと思います。

For / 医療法人・介護事業者

事業立ち上げの伴走

訪問看護、高齢者住宅、在宅医療クリニックの立ち上げを、制度・採用・運営の三点から設計し、開業後一年の運営まで伴走します。

For / 複数店舗を抱える経営者

採用・定着の仕組みづくり

求人票のリライトから面接設計、入職後六か月のフォロー体制まで。離職の原因は採用の前工程に潜んでいることが多いため、入り口と出口を一緒に直します。

For / 地域の医療ネットワーク

連携会議のファシリテーション

在宅医、訪問看護、ケアマネ、薬局の定例会を、単なる情報交換で終わらせない設計に。議題のつくり方から見直します。

For / 経営者個人

月一回の壁打ち面談

数字ではない、「本当はどうしたいか」を一時間、言語化していく場です。紙一枚に残して、次の決断を軽くします。
