OFFICIAL BASICS

そもそも「人材開発支援助成金」ってなに？

厚生労働省の公式情報では、事業主が従業員に職務関連の知識・技能を習得させるための訓練を計画に沿って実施した場合に、訓練経費や訓練期間中の賃金の一部を助成する制度です。

正式名称

人材開発支援助成金

従業員の人材育成・スキルアップを支援する国の助成金。irodoriの資料では「補助金」ではなく、原則この公式名に寄せて話します。

  • 申請主体はクライアント企業
  • 対象は雇用する労働者への訓練
  • 訓練経費と賃金の一部を助成
今回の本命

事業展開等リスキリング支援コース

新規事業の立ち上げ、事業展開、デジタル化・DX化などに伴い、新しい分野で必要になる知識や技能を習得させる訓練が対象になります。

  • AI活用を新しい業務能力として設計
  • HP更新・記録業務・資料作成へ接続
  • 令和8年度までの時限措置として確認
irodoriでの言い換え

制作物ではなく、使える人を育てる研修

助成金そのものを売るのではなく、クライアントの中にAIを使える人を増やす。ここをirodoriのAI人材育成パッケージとして見せます。

  • Claude Code / Codex導入研修
  • AI + 人の確認フローづくり
  • 研修後の運用定着は別枠で伴走

SOURCE CHECK

公式情報から押さえる最低ライン

  • 計画に沿った職業訓練として設計する。単なるツール契約や外注代行ではない。
  • 対象は、原則としてクライアント企業が雇用する労働者への訓練。
  • 事業展開等リスキリング支援コースでは、新分野・DX化等に伴う知識や技能の習得が軸。
  • 訓練開始前に職業訓練実施計画届を提出する必要がある。
  • 計画届を出しただけで支給が確定するわけではない。
  • 「訓練経費は無料になる」と見せず、支給申請までに事業主が負担する前提を置く。

POSITION

irodoriの役割は、外注先ではなく「超・提案型の左腕パートナー」へ。

HPを作る、資料を整える、記録業務を軽くする。その次に、クライアントの担当者が自分たちでAIを使える状態をつくる。ここに人材開発リスキリングの入口を置きます。

研修時間 10h+ 助成金活用を検討するなら、10時間以上のOFF-JT設計が基本ライン。
経費助成 75% 中小企業の経費助成率目安。最終判断は労働局・社労士確認。
提出期限 1M 訓練開始1か月前までの計画届を見込んで逆算する。
価格例 30万~ Claude Code / Codex導入研修は30万、60万、120万の3段階で仮置き。

SAFETY LINE

助成金の話は、言い方を最初から安全にする。

irodoriの信用を守るため、ここは強めに線引きします。人材開発支援助成金の採択や受給を売らない。社内人材育成の研修設計として話す。

言ってよいこと

  • AIを使って社内業務を改善する研修として設計できる可能性があります。
  • 条件が合えば、人材開発支援助成金の活用可能性を確認できます。
  • 研修費のサンプルとして30万・60万・120万円の設計案があります。
  • 最終判断は労働局・社労士等へ確認します。
  • irodoriは研修内容と業務改善の設計を支援します。

言わないこと

  • HP制作費がそのまま助成金で返ってくるとは言わない。
  • 75%返りますと断定しない。
  • ツール月額や個別開発費まで自動的に助成対象とは言わない。
  • 申請代行、採択保証、受給保証はしない。
  • AIツール導入だけを研修扱いにしない。

FACT CHECK / 2026-06-06

今回の松竹梅プランの制度前提

  • 対象想定は、人材開発支援助成金の「事業展開等リスキリング支援コース」。
  • 研修はOFF-JT 12時間で設計。10時間以上100時間未満の区分で見る。
  • 研修費は1名あたり30万円。梅1名、竹2名、松4名で金額を組む。
  • 中小企業の経費助成率75%で、梅22.5万円、竹45万円、松90万円を試算。
  • 賃金助成は1人1時間1,000円の可能性があるが、ここでは試算に含めない。
  • 計画届は訓練開始日の6か月前から1か月前まで。申請主体はクライアント企業。
  • 研修費は支給申請までに支払い完了が必要。最初から無料とは見せない。
  • 助成金の有無で価格を変える、不支給時に返金する設計は避ける。
  • 支給申請では受講料等の価格設定に関する疎明書が必要になる。
  • ツール月額、個別開発、社内サーバー接続、保守運用は研修費と分ける。

90 DAYS

まずは90日で、提案できる型まで育てる。

大きく売り出す前に、内部設計、候補選定、制度確認、初回提案の順に進める。

Phase 1 / 6月

内部設計

標準カリキュラム、禁止表現、1枚資料、確認リストを作る。

Phase 2 / 6-7月

候補選定

既存クライアントから2〜3件だけ、研修化できる業務を選ぶ。

Phase 3 / 7月

制度確認

社労士または労働局へ確認し、役割分担と価格分離を明文化する。

初回提案
