1. 報告概要

件名 健康管理支援室向け AI業務サポート研修 半年パックの実施検討
目的 訪問リマインド、顧客台帳、請求書、会議録整理などの事務作業を見直し、AI・デジタルツールを安全に使いながら業務負担を減らす。
実施案 2026年8月から2027年1月までの半年間。開始直後は週1回訪問し、その後は状況を見ながら訪問頻度を調整。オンライン相談は随時対応。
判断依頼 8月開始で検討を進めるか、受講者2名を選定できるか、補助金確認を社労士または労働局へ進めるか。

2. 概算見積書

研修名 産業保健業務のデジタル化・AI活用実践研修 半年パック
実施期間 2026年8月〜2027年1月
実施方法 訪問はまず週1回から開始。その後は状況を見て調整。訪問日以外はオンラインで随時相談。
助成対象研修 2名・合計12時間想定（雇用保険被保険者であることを要確認）
概算研修費 300,000円
経費助成見込み -225,000円
賃金助成見込み -24,000円
助成金活用後の法人負担見込み 51,000円

上記は中小企業扱い、経費助成75%、賃金助成1,000円/時を前提にした概算です。正式金額・助成可否・消費税の扱いは、正式見積および管轄労働局・社労士確認後に確定します。

3. 実施内容

主な対象業務 訪問リマインド、顧客台帳、請求書業務、会議録・面談メモ、送付状など
進め方 現場の実務を確認し、優先順位を決めながら、下書き・整理・確認しやすい形に整える。
安全面 個人情報・健康情報は人が管理し、AIは下書き・整理・抜け漏れ確認の補助に限定する。

4. 半年パックの流れ

8月 / 立ち上げ 週1回訪問から開始。AI利用の安全な線引き、優先業務の整理、進め方の確認。
9月 / 台帳整備 支援先情報・顧客台帳を整理し、次回訪問準備に使える形にする。
10月 / リマインド 訪問リマインド文面、Googleカレンダー・Gmail連携の考え方を整理。
11月 / 請求書整理 契約条件、交通費、イレギュラー請求など、請求書業務の確認ポイントを整理。
12月 / 記録整理 会議録・面談メモ・送付状を、共有しやすい記録に整える。
1月 / 定着確認 誤送信防止、人間チェック、運用ルール、次の改善候補を整理。

5. 補助金手続きの流れ

確認事項 申請主体の法人名、受講者2名の雇用保険加入状況、他業者の補助金利用予定を確認。
計画届 8月開始の場合、7月上旬までを目安に計画届を準備。
支給申請 研修終了後2か月以内に支給申請。助成金は審査後、法人さまへ後日支給。
次の進め方 8月開始で進めるか上長確認後、社労士または管轄労働局へ利用可否を確認し、正式見積へ進む。
