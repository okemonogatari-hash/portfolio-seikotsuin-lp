ゆかさんって、どんな人？

セッション中の言葉と、ストレングスファインダー34の結果から見えてきた、ゆかさんの輪郭です。

ゆかさんは、「今日が一番若い日」を本気で生きている保健師です。看護師として急性期7年、訪問看護11年、地域保健師3年、そして産業保健師4年目。21年の現場のあと、2025年に開業保健師として独立した方。

上位10資質を並べると、ある特徴がはっきり浮かびます。人間関係構築力4 × 戦略的思考力4 が同点トップ。"みんなが幸せであってほしい" と "新しいことを取り込むのに、不安よりワクワクが先に出る" が、ゆかさんの中で同じ強さで走っている並びです。

そこに 1位ポジティブ・2位適応性 が乗って、「めっちゃへこむけど、最終なんとかなるって思える」「死んでない、生きてる、最終そこに持っていける」という、20年現場を続けてきた根拠のない自信が出来上がっています。

そして7位最上志向。「自分たちにあったいい形の会社にしたい。普通のやつじゃなくて、二人の特性を合わせた、最大限活かせる会社」。2026年7月、合同会社設立予定。"2年で今の年収に並ぶ"を本気で言える人です。

CORE DISCOVERY

セッションで立ち上がった、3つの発見

朝のセッションで、ゆかさん自身の言葉から立ち上がった「これ、自分の場所かも」の瞬間。

CORE 01 — "自分を好きになる" より、もう少し近い距離

「自分を好きになる」じゃなくて「自分と仲良くなる」がしっくり来た

自己啓発本で「自分を好きになろう」を何度も読んで、何度も挫折してきたゆかさん。受け入れるところまでは行けたけど、「好きになる」までは届かなかった。そこにふっと出てきた言葉が、「自分と仲良くなる」。

「好きになるって、めっちゃハードル高いと思うんですよ。自分を愛せる、愛しそう、みたいな。無理と思う。だって嫌なとこしか今思い浮かばないしって思ってた。
"自分と仲良くなる"っていう言葉が、今すごいしっくり来たな。」

好きにならなくていい、けど、放っておかなくていい。「資質に名前をつけて、隣で暮らす」 の距離感が、ゆかさんが探していたちょうどいい温度でした。

CORE 02 — 二人で同時に、降りてきた事業

「保健師のAI人材を育てる」が、10秒で着火した

セッション終盤、おけちゃんが「AIと保健師さんのやり方を、ケースにしたら面白い」と振った瞬間、ゆかさんが秒で返した言葉。

「私たちの保健師のAI人材を育てる、それめっちゃいけますよ！
めっちゃいけますよね！ ニーズしかないから、ニーズしかない！ それやりましょう！」

そのまま具体策まで一気に出てきた：「まず私らでちょっと試してもらって、こういうところに仕事で使えるんやっていう、保健師の仕事の使いどころをどんどん教えてもらって」
これは「これあり」じゃなくて、「これやろう」モードに10秒で入った瞬間。新規事業の輪郭が、二人のあいだで同時に立ち上がりました。

CORE 03 — 軸になっている、娘の一言

「お母さんは今のままでいいやん」

コロナ禍でバーンアウト、家でワンワン泣いていた時期。リセットしたい、消えたい、と思っていたゆかさんに、当時小学4-5年生の娘さんがふっと言った一言。

「お母さんは今のままでいいやん、それで十分やんって言ってくれたんですよ。
それが、私の今の支え。"私、今のままでいいやん"って。それを娘に言われたんです。」

それまで看護師としては「申し上げなきゃ」と力が入っていた。でもこの一言を境に、面談でも「あなたはあなたで、十分やん」と返せるようになった。
ゆかさんが面談・コーチング・産業保健の現場で生み出している温度の原点は、ここに眠っています。

「自分を好きになる」じゃなくていい。
名前をつけて、隣で暮らすぐらいで、ちょうどいい。

YUKA'S EPISODE
ゆかさん本人の言葉：「仕事でもやっぱりみんなが幸せになってほしいんですよ。同じ働くなら楽しく明るく働きたいから。ネタ提供するとか、常にみんなが笑えるようなことは、別に意識してるわけじゃない、それが好きだからやってる」

ただし、本人は「めっちゃ悩むしへこむし、ネガティブな面めっちゃある」と即座に補足。「悩まない人」じゃなくて、「みんなを楽しく明るくしたい」が止まらない人。落ち込んだあとに「ネタ見できる」「最終なんとかなる」と立ち戻れるのも、この資質の働き方です。
