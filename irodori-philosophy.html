Our Value

クライアントに提供する
5つの価値の柱

バリューとは、IRODORIがクライアントに提供する具体的な価値のこと。
理念やビジョンを実現するために、実際にどのような価値を届けるのかを明確にしたものです。

VALUE 01

気づきをもたらす
外の視点

客観性やプロ目線で、自社では気づかない魅力や課題を発見し提供します。

VALUE 02

やるべきことに集中できる時間を創出

ドラえもんのように、クライアントの時間を生み出し、やるべきことに集中できる環境を実現します。

VALUE 03

社会にやさしい
仕組みづくり

子育て中の優秀な人材を活用し、企業の人件費も軽減する、社会にやさしい仕組みです。

VALUE 04

"最初からプロに頼む"を当たり前に

手探りで進める無駄をなくし、プロの力で最短距離での目標達成を支援します。

VALUE 05

"想い"を伝わる
カタチに

デザイン、ライティング、事務経理など、各分野のプロの力であなたの魅力を彩ります。

Core Values

7つのコアバリュー

コアバリューとは、IRODORIのメンバー全員が大切にする行動指針・価値観のこと。
日々の判断や行動の基準となり、チームの文化を形成します。
1

縁ある人々に最上級の愛情と思いやりを持って接す

相手目線

押し付けではなく、動機(想い)を聴きながら相手目線で寄りそう

クライアントは推し！

愛を言葉と行動で示す

小善は大悪に似たり

表面的な優しさだけで本質を逸らさない

Do 実践すること
  • 助け合い
  • 相手理解を優先
  • 必要な厳しい提言も愛
Don't 避けること
  • 事務的
  • 自己本位
  • 形だけの親切
2

これまでの人生・今起きている出来事の、全ての物事に感謝をする

一期一会と利他の精神

一つひとつの出会いを大切に

原因自分論

起きた出来事の責任を自分に置く

「これだけある」の視点

「これしかない」ではなく前向きに捉え直す

Do 実践すること
  • 1メッセージに、1ありがとうを添える意識！
  • 指摘は期待の裏返し（クレームはギフト）
  • 違いは間違いじゃない！柔軟に受け止める
Don't 避けること
  • 協力を当たり前とみなす
  • 感謝を忘れる
  • 他責思考・他社/他者否定・言い訳
3

真心を込めて、相手の気持ちに立ち、常に先回りで気配りを行う

そこまでやる？がスタンダード

期待を超える気配りを当たり前にする

配慮はしても遠慮はしない

相手のために必要なことは言い切る

相手の1番の応援団

勝たせるために全力で支える

Do 実践すること
  • すみません→ありがとうに変える
  • 共感したうえでの提案
  • 倒れたら支える！人として当たり前の助け合い
Don't 避けること
  • 過剰に気を遣いすぎて疲弊
  • 押し付けがましい
  • 自分本位
4

常にご機嫌かつ笑顔で周りを明るく照らす

ニコ・キビ・ハキ

いつもニコニコ笑顔で、キビキビと動き、ハキハキと受け答えする

同じ波動の人が集まる

ポジティブなエネルギーの人の周りにはポジティブな人が集まる

解釈は無数

事実は1つだが、解釈は無数。自分の捉え方次第

Do 実践すること
  • 自分から挨拶
  • ポジティブな言葉選び
  • 笑顔を先出し
Don't 避けること
  • 感傷的な悩み
  • 過去への固執
  • ネガティブな態度
5

ごまかしや言い訳はせず、時間や約束など決めたことを誠実に守る

嘘をつかない

事実に対してまっすぐ向き合う

先回りコミュニケーション

聞かれる前に共有・確認する

動機善なりや、私心なかりしか

動機と目的を常に問い直す

Do 実践すること
  • 言行一致
  • 誠実な謝罪→改善
  • 約束を守る
  • 愚痴や不平不満だけでなく、一緒に改善策を考える
Don't 避けること
  • 値引き前提
  • 安請け合い
  • 雑な仕事
6

仕事に熱い情熱を持ち、誰よりも本気で取り組む

プロの誇り

どんな仕事にも誇りを持って取り組む

自燃する

人に言われたからではなく、自分から仕事の面白さやワクワクを見つける

持続可能な熱量

燃え尽きず、長く走り続ける

Do 実践すること
  • 小さな仕事にも心を込める
  • 仕事を楽しむ
  • 最高を追求
Don't 避けること
  • 不燃性
  • 空回り
  • 遠慮しすぎ
7

常に学び続け、高みを目指して挑戦し続ける

現状維持は退化の始まり

立ち止まった瞬間に後退していることを知る

GoodはGreatの敵

「そこそこ」に満足しない

完璧は無い。常に改善

今日の最高を、明日さらに超えていく

Do 実践すること
  • 優先順位の明確化
  • ライバルは他人ではなく、自分
  • 学んだことをすぐ実践
Don't 避けること
  • 「駆け出しなので」は使わない（クライアントにとって私たちはプロ）
  • 「なんとなくいい感じ」で満足
  • 学んだだけで実践しない
