Chapter 01 / Context

なぜ今、この事業なのか。

CW・個人案件の積み上げは続くけれど、
「安定した事業の柱」にはなりきれない。
一方で、2027年3月末までの「政策の波」が目の前にある。

CW依存からの脱却が必要
案件単価は頭打ち、継続性が読めない。「次の1件」をずっと追い続けるモデルは疲弊する。
企業向け継続売上の不在
¥300,000/社の安定単価、事例が積み上がれば次の営業資産になるモデルが、今のラインナップにはない。
政策の波は時限付き
人材開発支援助成金は2027年3月末まで。国が「AI人材育成」に予算を張っている今が、最も追い風が吹く時期。
The Window of Opportunity
今動けば、制度の追い風に乗れる。
動かなければ、波はただ過ぎていく。
制度変更は2027年4月以降。今のうちに事例・実績・ノウハウを資産化できるのは、これから約11ヶ月。

Chapter 03 / Discovery

ファクト検証で見えた、
3つの決定的な発見。

2026年4月17日、全CxO合同で制度要件を精査した結果、
「うちが勝てる構造」がくっきり見えた。

01
社内研修はNG、外部委託が制度要件。
「社内講師による研修は原則対象外、外部講師委託が助成金要件」— これが制度設計の核。つまり補助金を取りに行く企業は、必ず外部の研修会社に頼むしかない。市場は国の制度側から作られている。
外部委託マーケット確定
02
申請書は公式テンプレ公開。企業が自分で申請できる。
厚労省が申請書類一覧を公開済み。社労士に代行させる必要がない。つまり「うち経由で研修受注 → 申請は企業自身」の完結型モデルが成立する。社労士パートナーは必須じゃない。
社労士パートナー不要
03
Claude Codeで申請書作成支援ができる。
うちの本業（AI研修）と、企業が自分で申請書を書く作業が、同じ道具（Claude Code）でつながる。「研修で教える技術 × 申請書作成支援」のダブルバリュー。他のAI研修会社にはない独自構造。
差別化の核
この3つが揃ってる時点で、
うちがやらない理由が、ひとつも無い。

Chapter 05 / Product

商品構造は、2層で設計する。

Part Aは補助金対象の研修本体。
Part Bは差別化の核 — 他社にない「申請サポート」パッケージ。

MAIN PACKAGE
AI研修10時間パッケージ
¥300,000/社
Part A 研修本体(補助金対象)
制度要件をクリアする10時間の研修プログラム
  • eラーニング 2時間 — AI基礎 / Claude / プロンプト基礎
  • 対面・オンライン 8時間 — ハンズオン / 実務ツール / 業務適用設計
  • 対象：正社員 / 補助金要件クリア

企業が自分で申請書を完成させるための独自パッケージ
  • 申請書テンプレ集 — 厚労省公式様式準拠
  • Claude Code用プロンプト集 — 申請書作成補助（うち独自）
  • 記入例・Q&A集
  • オンライン質問1回
→ 社労士不要。企業が自分で申請まで完結できる構造が、うちの最大の武器。

Chapter 09 / Risk

見えているリスクと、
その対策。

行きましょう、と言う前に、ちゃんとリスクも見える化しておく。
外部監査（外部監査）からも本件は「本気で取りに行ってOK」の評価を獲得済み。

Risks — 見えている懸念
起こりうること
  • 2027年4月以降の制度変更で補助率・要件が変わる可能性
  • 「補助金ありき」の商品設計だと、制度が動くと事業が揺れる
  • 社労士法への抵触（代筆・代理申請）リスク
  • Phase 1で1社目が取れなかった場合、事例資産が作れない
Actions — 織り込み済みの対策
うちはこうする
  • 研修そのものの質を磨く = 補助金が無くても売れる商品に育てる
  • 業界特化・事例資産を並走で構築し、単価×継続性で勝負
  • 申請代行は業として行わない。研修 + ツール + 情報提供までに線を引く
  • Phase 1は代表の既存経営者接点から攻める（信頼の上で提案）

この事業、本気で取りに行ってOK。2027年3月末までの期間限定だから、Phase 1で1社成約 → Phase 2で加速の流れは妥当。ただし警告：補助金制度が変わっても事業が持続する商品力（研修そのものの質）を並走で磨くこと。
