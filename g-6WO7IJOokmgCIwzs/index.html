🌱 はじめに

清美環さんへ。先日のお打ち合わせで伺った「現場写真をエクセルの写真帳に貼り付ける作業を、もっとラクにしたい」を受けて、土台を作ってお試し版として用意したものです。

このページは、道具一式の中身・できること・受け取り方・実際の動きが、ぜんぶ1ページで分かるようにまとめてあります。気になるところからどうぞ。

📦 名前： photo-album
🔖 これは何？ Claude Code（AIに自分用の道具を作らせる仕組み）に、写真帳づくりの進め方を覚えさせた"道具一式"です
🎯 想定： Google等の外部サービスは使わず、お使いのパソコンの中だけで完結します
⚠️ 大事なお知らせ：入れたら必ず一発で完璧に動く、というものではありません。
この道具一式は、Claude Code（AIエージェント）が写真帳づくりを進めるための「手順・部品・確認方法」をまとめたものです。パソコン環境や写真の撮り方、ファイル名の違いによって、最初は調整が必要になることがあります。エラーが出た場合も、画面の内容を見ながら、AIエージェントが原因を確認し、必要な修正案を出しながら進めていく前提の叩き台です。

📌 こちらでも何度もチェックしてお渡ししていますが、次のような理由で「最初から完璧に動く」とは限りません。正直にお伝えしておきます。

  • パソコンの状態が、こちらと清美環さんとで違います
    Mac／Windowsの違い、入っているPythonの種類やバージョン、必要な部品があるかどうか——これが少し違うだけで、最初の起動で止まることがあります。
  • Claude Codeが入っていても、つないでいる道具の組み合わせが違います
    同じClaude Codeでも、人によって設定や接続している道具が違うので、こちらと同じ動きになるとは限りません。
  • 現場のお写真は、毎回少しずつ違います
    ファイル名のつけ方・写真のサイズ・中に焼き込まれた文字などが現場ごとに違うため、最初の試運転で「ここだけ自動で判定できなかった」という箇所が出ることがあります（黄色いマスでお知らせします）。
  • 叩き台版なので、現場ルールに合わせた追い込みはこれからです
    写真の高さの単位や、位置コードの決め方など、清美環さんの現場のやり方をうかがいながら、これから一緒に整えていく前提です。

→ 止まったときは、画面の様子を Claude Code に見せながら、一緒に原因を確認して進めるのが、いちばん早くて確実です🌱

📋 どんなことができるか

現場で撮ったお写真のフォルダを渡すだけで、写真帳のエクセル（写真帳_出力.xlsx）が自動で組み上がります。手で写真を貼り、サイズを揃え、文字を打ち込んでいた部分を、道具が代わりに整えます。

  1. 写真の仕分け（取付管系／異常箇所系を見分ける）
  2. 配置（写真帳の枠に、決まったレイアウトで並べる）
  3. サイズ揃え（写真を枠に合わせて自動リサイズして埋め込む）
  4. 文字の書き入れ（「No.」「カテゴリ」「位置」「J番号」「メーター数」を記入）
  5. 確認ポイントの見える化（自動で判断できない所は黄色いマスで「ここ見て」と示す）

大事なルール

  • お写真は外に出ません — すべてパソコンの中だけで処理します。自動でどこかに送信されることはありません
  • ごまかさず、黄色で見せます — 読み取れない所をそれっぽく埋めず、確認してほしい場所を黄色で残します
  • 100%の完成保証ではありません — 環境差や写真の状態によって、AIエージェント側で確認・修正しながら進めることがあります
  • まずは叩き台 — いきなり完璧な全自動ではなく、「実際の現場写真でどこまで使えそうか」を確かめるための初版です

🛠 どうやって作ったか

大きく3段階で作りました。

① お預かりした実物を読み解いた

お預かりしたエクセルの写真帳テンプレ（K1903042）と、現場写真10枚の中身（焼き込み文字・撮影角度・構図）を1枚ずつ確認し、「どの写真をどこに、どんな文字と一緒に貼るか」のルールを抽出しました。

② Claude Codeで道具一式に組み上げた

抽出したルールをもとに、環境チェック・写真帳生成・手引きをひとまとめのスキルにしました。出力は、Google等を介さずパソコンの中だけで完結する形（エクセル直）にしています。

③ 素人マシンでも動くよう「環境ドクター」を付けた

AIに詳しくない方のパソコンでも詰まらないよう、初回に環境を自分で診断して、足りないものをやさしく案内する 環境ドクター（setup_check.py） を入れてあります。ここが、この道具一式の特徴です。

🏗 中身の構成

道具一式は1つのフォルダで完結します。中身はWindowsでも扱いやすいようにまとめています：

🔎 写真の判定ルール

道具が写真帳を組むときに使っている、現時点のルールです。これが「写真の自動仕分け」の核です。

取付管 or 異常箇所の見分け方

  • ファイル名の末尾「S」→ 取付管・精密写真
  • ファイル名の末尾「F」→ 異常箇所・全体写真
  • 写真内の文字読み取りは、今後の改善ポイントです。このお試し版では、まずファイル名と固定配置で確認します

写真の組み合わせ・配置

  • 1組＝「全体写真」＋「アップ写真」の2枚
  • 1ページ＝上段に全体・下段にアップ／左右2ペアの構成
  • 写真がない枠には「余白」と書き入れます

文字の書き入れ方

  • 取付管の枠：「No.」「取付管」「位置」
  • 異常箇所の枠：「No.」「異常箇所」「No.X」「浸入水b」「J番号」「メーター数」
  • お預かりしたエクセルの書き方を、そのまま踏襲します

📝 サンプルで作るとこうなる

⚠️ これはお試し出力です。 同梱のサンプル写真10枚（K1903042）だけを材料に、道具が実際にどう動くかを見ていただくためのものです。

入力： サンプル写真10枚（取付管系7枚＋異常箇所系3枚）
所要時間： 数秒
出力： 写真が貼り込まれた3ページ構成の写真帳エクセル（画像はファイルの中に実体として埋め込み。ExcelでもNumbersでも開けます）

⬇️ 出来上がりサンプルを見る（サンプル写真帳.xlsx／約1.2MB）

↑ ダウンロードして ExcelまたはNumbers で開くと、実物の見た目が確認できます。

ページ1：取付管L-1（全体／アップ）— 右ペアに2枚、左ペアは余白
ページ2：取付管L-1（左ペア）＋ 異常箇所 No.1（J1・3.67m／右ペア）
ページ3：異常箇所 No.2（J2・6.02m）＋ No.3（J3・8.33m）

このように、同じ取付管L-1を角度違いの4枚で1セットにする構造まで再現しています。読み取れない所が出たら、その枠は黄色いマスで「確認してください」と残します。

📂 受け取り方

今回はWindowsでのご利用を想定しています。ZIPをダウンロードして、Claude Codeのスキル用フォルダに置けば使える形です。

ただし、ZIPを入れた時点で、どんなパソコン・どんな写真でも必ず完成することを保証するものではありません。 スキルは、AIエージェントに作業の進め方を渡すためのものです。実際には、同梱サンプルや本番写真で動きを確認しながら、環境差・ファイル名・写真の状態に合わせて調整していきます。

🌟 大事な前提：AIエージェントと確認しながら育てる道具です

下の手順どおりに進めても、パソコンの状態や写真フォルダの中身によって止まることがあります。その場合は失敗ではなく、Claude Code に画面の内容を見せて、原因を確認しながら直していく流れです。

たとえば、こんなふうに話しかけるだけで大丈夫です：

  • 「Windowsで、ダウンロードした photo-album.zip をClaude Codeのスキルとして使えるようにしてください」
  • 「画面にこんなメッセージが出ました（スクショを貼る）。どうしたらいいですか？」
  • 「ここの意味が分かりません。やさしく教えてください」

「入れたら100%自動で完成する道具」というより、「AIと確認しながら現場用に整えていく土台」です。分からないところは Claude Code に状況を渡しながら進めてください。

✅ Windowsで設置する

止まった場合は、Claude Codeにエラー内容を見せて原因確認します

⬇️ photo-album.zip をダウンロード（約1.6MB）

  1. ダウンロードしたZIPを右クリック → すべて展開します（photo-album フォルダができます）
  2. Claude Codeに、下の文章をそのまま貼ります
  3. 初回だけ、環境ドクターで準備を確認します： 
    py -3 setup_check.py
  4. Claude Codeを開いて、合言葉を話しかける： 
    写真帳作って

※ py -3 が動かない場合は、Claude Codeが python で動くか確認して進めます。

Claude Codeが「了解、まず環境を確認しますね」「写真フォルダはどれですか？」のように返してくれたら、確認の流れに入れています。

💬 詰まったとき： 変なメッセージが出たり、うまく動かないときは、その画面のスクショや表示文を Claude Code に見せて、原因を確認しながら進めます。
⚡ Claude Codeに設置確認を頼みたい方向け（環境によっては安全機構で止まります）

下の指示文をコピーして、Claude Codeに貼り付ければ、Windows向けに解凍〜設置〜確認まで進めてくれます。

※ バージョン・設定によっては「外部からの取得は実行できません」と止まることがあります（安全のための正常な動作）。その場合は上の手動ルートで入れてください。

Windowsで、ダウンロードした photo-album.zip を解凍して、
Claude Codeのスキルとして使えるようにしてください。

手順：
1. ダウンロードフォルダにある photo-album.zip を展開
2. %USERPROFILE%\.claude\skills\photo-album に置く
3. py -3 setup_check.py sample を実行
4. py -3 build_album_xlsx.py を実行
5. py -3 verify_album_xlsx.py で、Excel内に写真が埋め込まれているか確認
6. 結果をやさしく教えて

🚶 最初の使い方の提案

最初は、いきなり本番運用に入らず、次の順で少しずつ確認するのがおすすめです：

  1. まず同梱のサンプル写真で、写真帳エクセルが作れるか見てみる
  2. つぎに本番の写真フォルダを1つ選んで、写真帳を作ってもらう
  3. 出てきたエクセルを開いて、写真の入り方と黄色マスを見る
  4. 「現場のやり方とここが違う」「この並びはこうしたい」を教える
  5. その感想を道具一式に反映して、清美環さん専用としてどんどん育てる
