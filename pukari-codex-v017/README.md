作成日: 2026-05-31

## 目的

飯田さん経由の司法書士事務所HP制作相談について、甲斐さん追加質問「ページ中程に横スクロールの解決事例を入れられるか」に対する確認用デモを、クライアント/訪問者向けの自然な「ご相談事例」セクションとして整える。

制作側の検証ではSWELL系の写真つきカルーセルを参考にするが、画面上には「SWELL風」「確認用デモ」「技術的にできること」などの制作側メモを出さない。

v0.1.6でループ切り替え時のガクつきを抑え、v0.1.7では画面上の制作側説明を削って、訪問者向けの文言へ変更する。

## 方針

- 拓実リーガルの「信頼感・相談導線・業務整理・更新情報」の構成を参考にする
- 色はプカリのロゴに合わせて青×オレンジへ置き換える
- WordPress前提に固定せず、コラム運用確認後に技術選定する
- まずはHTMLモックとして、打ち合わせで方向性確認できる状態にする
- 解決事例は実案件ではなく、守秘性に配慮したモデルケースとして表示する
- 今回は「訪問者が自分に近い相談例を見つけやすい、ご相談事例セクション」として自然に見えるかを見るデモ。自主管理や保守範囲の話は別フェーズとして扱う
